【软件说明】
 * 软件名称：   SBUS航模遥控器数据解析例程 
 * 主要功能：   HOTRC航模遥控器SBUS数据解析，并通过串口输出
               支持所有型号，不同遥控器通道数不同，最多支持10个通道

【硬件说明】
 * 硬件平台：   OpenCTR H60 V3.2
 * 处理器：     STM32F407VET6
 * 高速晶振：   8MHz
 * 总线时钟：   系统时钟 = SYCCLK = AHB1 = 168MHz
 * 硬件接口：   参考开源硬件原理图
   
【其它说明】
 * 为了获取更好的阅读体验，建议将Tab size设置为4
 * 通过Edit菜单下，Configuration子菜单设置窗口的Editor标签页的“Tab size”选项设置


【硬件连接】
   
   SBUS信号经过转换后  ----  PD2（UART5 - RX）
