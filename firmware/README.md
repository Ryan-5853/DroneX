# DroneX 飞控固件子系统

基于 HAL 库的飞控固件与外围驱动，面向 DroneX 推力矢量可回收火箭式无人机。

## 简介

本子系统基于 **STM32H743** 与 **STM32CubeMX/HAL**，从零编写飞控固件，实现：

- 实时控制循环（200–500 Hz）
- IMU 姿态估计（互补滤波或 EKF）
- 电调与云台电机驱动
- 遥控接收与串口遥测
- 安全逻辑（低电压、失控、急停）

## 参考系与物理量单位规定

    世界参考系：在初始化时与机体参考系对齐，随后保持与地面相对位姿不变
    机体参考系：俯视飞控板正面，向上为y轴正方向，向右为x轴正方向，垂直飞控板正面向上为z轴正方向，原点在机体质心
    传感器参考系：近似认为与飞控板姿态相同，但是原点在飞控板中心
    矢量单元参考系：俯视矢量舵盘，使两突出舵臂分别指向平面直角坐标系的x、y正方向，称为x臂和y臂，垂直矢量舵盘朝上为z正方向
                    两舵机分别称为x舵机、y舵机，当舵机臂水平于地面时，认为舵机打角为0，向上为正，向下为负，按照对称限幅归一化到-1~1
                    矢量单元朝向以roll/pitch表示，其中roll为绕y轴的转角，pitch为绕x轴的转角，遵循右手螺旋定则。即x舵机正向打角时，x臂上抬，矢量单元绕y轴逆向旋转；y舵机正向打角时，y臂上抬，矢量单元绕x轴正向旋转；

    长度、位移均以m为单位；质量以kg为单位；力以N为单位；角度以360分度为单位；时间以s为单位
