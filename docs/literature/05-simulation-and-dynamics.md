# 仿真与动力学建模 — 文献摘编

支持 DroneX **仿真子系统**（MATLAB/Simulink）中的刚体动力学、坐标系约定、推进器与云台模型，以及控制器在环仿真 (CIL)。

---

## 1. 与 DroneX 的对应关系

| 主题 | 在项目中的作用 |
|------|----------------|
| 坐标系与姿态表示 | 惯性系（地面、Z 向上）、机体系（重心）、推力/云台系，与 PLAN.md 一致 |
| 平动方程 | m·a = F_thrust + m·g + F_disturbance |
| 转动方程 | I·ω̇ + ω×(I·ω) = τ_thrust + τ_disturbance |
| 推进器与云台模型 | 推力 T(Ω)、共轴干扰；云台角 (α, β) 与电机动力学、限位 |
| CIL | 与真实控制器代码或等价逻辑在环，验证参数与鲁棒性 |

## 2. 推荐文献要点

### 2.1 飞行器动力学建模与仿真（Zipfel）

- **书名**：*Modeling and Simulation of Aerospace Vehicle Dynamics*  
- **与项目关系**：六自由度刚体方程、不同坐标系（NED/ENU/体轴）的定义与转换，以及欧拉角/四元数微分。
- **可借鉴**：  
  - 惯性系下位置/速度微分；体轴系下角速度与欧拉角速率关系。  
  - 转动惯量矩阵 I 在体轴系下的常数假设；外力/力矩在体轴系下的表达。  
  - 与 PLAN.md 中“惯性系 Z 向上、机体系原点在重心”的约定对应。

### 2.2 刚体与多体动力学（Jazar）

- **书名**：*Advanced Dynamics*  
- **与项目关系**：欧拉角奇异点（万向锁）、四元数微分方程与归一化，多体系统中铰链与约束。
- **可借鉴**：  
  - 俯仰角 θ 接近 ±90° 时欧拉角速率退化，仿真中建议用四元数积分。  
  - 云台相对机体的转动可建模为多体铰链（俯仰轴、滚转轴），若用 Simscape Multibody 可对应为 Revolute 关节。

### 2.3 Aerospace Blockset / Simscape（MathWorks）

- **与项目关系**：PLAN.md 指定 MATLAB（含 Simulink）为仿真工具，可选 Simscape Multibody 做机械/接触。
- **可借鉴**：  
  - **Aerospace Blockset**：6DoF 刚体块、重力、风模型；自定义推力/力矩接口接推进器+云台模型。  
  - **Simscape Multibody**：机架、云台、推进单元为刚体，铰链与驱动；适合校核机构与耦合动力学。  
  - 控制器在环：Simulink 中嵌入与飞控等价的离散控制器（或自动生成 C 代码做 SIL）。

### 2.4 四旋翼 Simulink 建模（Luukkonen, Aalto）

- **题目**：Modelling and Control of Quadcopter  
- **与项目关系**：完整的 Simulink 框图、参数表、线性化与控制器设计，流程可迁移到 TVC 无人机。
- **可借鉴**：  
  - 状态变量选取：位置、速度、姿态（四元数或欧拉角）、角速度。  
  - 推进器子块：指令 → 推力/力矩；在 DroneX 中改为 T + (α, β) → F_thrust, τ_thrust。  
  - 验证标准：阶跃响应、稳定裕度、扰动抑制。

### 2.5 坐标系与标准

- **NED / ENU**：惯性系 Z 向上即 ENU 一类；文献与仿真中需统一符号（如 X 前、Y 左、Z 上）。  
- **PLAN.md**：惯性系地面固定 Z 向上；机体系原点在重心；推力系随云台偏转。建议在仿真文档中写清各系定义与转换矩阵。

## 3. 与 PLAN.md 的衔接

- **仿真子系统**：刚体动力学、推进器（推力–转速、共轴干扰）、云台动力学（俯仰+滚转、电机响应、限位）、CIL；可选 Simscape Multibody。  
- **输出**：验证过的控制参数、轨迹规划结果。  
- **开发阶段 P0**：动力学推导、仿真环境搭建、控制器初步验证 → 仿真报告、控制参数初值。

---

## 4. 仿真实现建议

- **状态向量**：位置 p（惯性系）、速度 v（惯性系或体轴）、四元数 q、角速度 ω（体轴）；微分方程见 Zipfel/Stevens。  
- **推力与力矩**：由 T 和云台角 (α, β) 得到 F_thrust 在体轴系下的分量，再算 τ_thrust = r_cp × F_thrust；r_cp 为推力作用点在体轴系下相对重心的位置。  
- **云台模型**：二阶或一阶+限位，输入为期望 (α_d, β_d)，输出为实际 (α, β)；与真实电机带宽一致便于 CIL 可信度。  
- **验证**：悬停平衡、阶跃姿态/位置、风扰动、降落段轨迹；与后续实机 P2–P4 阶段对照。
