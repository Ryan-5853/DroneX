# 倒立摆与欠驱动系统 — 文献摘编

DroneX 的**重心高于推力作用点**，开环为**固有不稳定**，力学上等价于倒立摆；控制思路可借鉴倒立摆与欠驱动机械系统文献。

---

## 1. 与 DroneX 的对应关系

| 倒立摆/欠驱动系统 | DroneX |
|-------------------|--------|
| 摆角 θ | 机体俯仰/滚转角 |
| 摆的支点加速度或力矩输入 | 云台偏转产生的力矩 + 推力水平分量产生的加速度 |
| 欠驱动（少于一维一电机） | 推力幅值 T + 二轴云台 (α, β) 同时决定平动与转动，存在耦合与分配 |

---

## 2. 推荐文献要点

### 2.1 反馈系统与稳定性（Åström, Murray）

- **书名**：*Feedback Systems: An Introduction for Scientists and Engineers*  
- **与项目关系**：倒立摆作为经典例题，说明线性化、极点配置、LQR 与稳定性判据。
- **可借鉴**：  
  - 在平衡点（竖直向上）附近线性化，得到 θ̈ = (g/ℓ)θ + (1/(mℓ²))τ 形式。  
  - DroneX 中转动方程在悬停姿态附近线性化后，同样可得类似“不稳定极点 + 控制输入”的形式，便于设计 PID/LQR。

### 2.2 欠驱动机械系统（Fantoni, Lozano）

- **书名**：*Non-linear Control for Underactuated Mechanical Systems*  
- **与项目关系**：推力矢量机同时要稳定姿态（转动）和位置（平动），属于欠驱动、耦合系统。
- **可借鉴**：  
  - 级联控制：内环快速稳定姿态（角速度/姿态角），外环位置/速度产生“期望姿态或期望推力方向”。  
  - 与 PLAN.md 中“串级 PID，内环角速度、外环姿态角”一致；外环再套位置/速度环。

### 2.3 Acrobot 起摆与稳定（Spong）

- **题目**：The Swing Up Control Problem for the Acrobot  
- **与项目关系**：从任意初始角“起摆”到倒立位置并稳定，类比 DroneX 从大倾角恢复或起飞瞬态。
- **可借鉴**：  
  - 能量/基于能量的起摆律；稳定阶段切换为 LQR 或 PID。  
  - 若 DroneX 需从较大初始倾角（如放置倾斜）自主扶正，可参考切换逻辑与安全边界。

### 2.4 微四旋翼（Bouabdallah 等）

- **题目**：Design and Control of an Indoor Micro Quadrotor  
- **与项目关系**：四旋翼也是欠驱动、强耦合；部分文献讨论重心与推力平面不重合时的稳定性。
- **可借鉴**：  
  - 角速度内环 + 姿态外环的 PID 结构及参数数量级。  
  - 若引入“重心偏离几何中心”的模型，与 DroneX 的“重心在推力之上”在数学上同属一类耦合。

---

## 3. 与 PLAN.md 的衔接

- **倒立摆特性**：文档已写明“重心在推力之上时，推力偏转产生的力矩可稳定姿态，但开环系统不稳定，需闭环控制”。  
- **控制结构**：姿态控制采用“串级 PID 或 LQR，内环角速度、外环姿态角”；位置控制为外环位置/速度 PID；控制分配将期望力/力矩映射到 T 与云台角 (α, β)。  
- **降落逻辑**：悬停 → 减速下落 → 触地检测/缓冲，可与“稳定平衡点”的保持与安全切换一起设计。

---

## 4. 仿真与调参

- MathWorks “Inverted Pendulum” 示例：线性化、LQR 设计、Simulink 实现，可直接对照 DroneX 在 Simulink 中的姿态环设计。  
- 调参顺序建议：先在仿真中稳定内环角速度，再外环姿态角，最后位置环；实机时同样由内而外、逐步放开激励。
