# 飞行控制与状态估计 — 文献摘编

面向 DroneX 的**姿态控制**、**位置/速度控制**、**控制分配**以及**IMU 姿态估计**（互补滤波或 EKF），便于算法设计与 C 代码实现。

---

## 1. 与 DroneX 的对应关系

| 主题 | 在项目中的作用 |
|------|----------------|
| 姿态表示与欧拉方程 | 机体系、惯性系、角速度与姿态角微分方程，与 PLAN.md 坐标系一致 |
| 姿态控制 (PID/LQR) | 内环角速度、外环姿态角，输出期望力矩 → 控制分配 |
| 位置/速度控制 | 外环 PID，输出期望加速度或期望推力方向 |
| 控制分配 | 期望力 F_d、力矩 τ_d → 推力 T + 云台角 (α, β) |
| 状态估计 | IMU 陀螺+加速度计 → 姿态（互补滤波或 EKF），角速度可直接用陀螺 |

## 2. 推荐文献要点

### 2.1 飞行器控制与仿真（Stevens 等）

- **书名**：*Aircraft Control and Simulation*  
- **与项目关系**：欧拉角/四元数、刚体转动方程、PID 与 LQR 设计流程，以及仿真框图。
- **可借鉴**：  
  - 欧拉角微分与角速度关系：φ̇, θ̇, ψ̇ 与 p, q, r 的转换；奇异点（θ = ±90°）及四元数替代。  
  - 串级控制：内环角速度阻尼，外环姿态角跟踪；LQR 权重选取与仿真验证。

### 2.2 先进飞行器/航天器控制（Tewari）

- **书名**：*Advanced Control of Aircraft, Spacecraft and Rockets*  
- **与项目关系**：推力矢量飞行器的姿态回路、制导与控制的层级划分。
- **可借鉴**：  
  - 姿态误差（四元数或欧拉角）到角速度指令、再到力矩指令的映射。  
  - 与 PLAN.md 中“控制分配：期望力/力矩 → T + (α, β)”一致的整体结构。

### 2.3 互补滤波在 SO(3) 上的形式（Mahony 等）

- **题目**：Nonlinear Complementary Filters on the Special Orthogonal Group  
- **与项目关系**：仅用 IMU 时，陀螺积分易漂移，加速度计可提供重力方向约束；该文给出在 SO(3) 上严格、全局的互补滤波。
- **可借鉴**：  
  - 嵌入式上计算量适中，无需线性化；可得到一致、无奇异的姿态估计。  
  - 与 EKF 相比，实现简单，适合首版飞控；若需更高精度再考虑 EKF。

### 2.4 最优状态估计与 EKF（Simon）

- **书名**：*Optimal State Estimation*  
- **与项目关系**：若采用 EKF 做姿态估计，需处理四元数约束、观测模型（重力方向、磁强计等）。
- **可借鉴**：  
  - 四元数 EKF 的误差状态表示（避免四元数归一化与约束问题）。  
  - 观测方程：加速度计在静止/缓动时 ≈ 重力在机体系下的投影；磁强计可辅助偏航。

### 2.5 四旋翼建模与控制（Bresciani, Lund）

- **题目**：Modelling, Identification and Control of a Quadrotor Helicopter  
- **与项目关系**：从建模、参数辨识到控制器设计的完整流程，可迁移到 TVC 无人机。
- **可借鉴**：  
  - 系统辨识：阶跃/扫频激励、传递函数或状态空间辨识。  
  - 控制结构图、调参顺序（内环→外环）、鲁棒性考虑。

## 3. 与 PLAN.md 的衔接

- **姿态控制**：串级 PID 或 LQR，内环角速度、外环姿态角。  
- **位置控制**：外环位置/速度 PID。  
- **控制分配**：期望力/力矩 → T + (α, β)；需考虑 T 的范围与云台角限位，必要时加入伪逆或优化分配。  
- **下位机**：IMU（MPU6050/BMI088）、姿态估计（互补滤波或 EKF）、实时控制循环 200–500 Hz。

---

## 4. 实现建议

- **姿态表示**：推荐四元数存储，避免欧拉角奇异；显示或调试时可转换为欧拉角。  
- **控制周期**：内环（角速度）高于外环（姿态、位置）；与 PLAN.md 中 200–500 Hz 一致，内环可取 500 Hz，外环 100–200 Hz。  
- **控制分配**：二轴云台 + 单推力源时，(F_d, τ_d) 到 (T, α, β) 的映射有解析形式，注意 T > 0 与云台机械限位约束。
