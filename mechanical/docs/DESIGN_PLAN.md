# DroneX 机械设计计划

> 依据 [PLAN.md](../../../docs/PLAN.md) 第 5.3 节及第 7 节细化方向编写。

## 一、设计约束与目标

### 1.1 力学约束

| 约束项       | 要求                                      |
| ------------ | ----------------------------------------- |
| **重心位置** | 重心必须高于推力作用点（倒立摆稳定所需）  |
| **推力点**   | 与共轴推进器中心、云台旋转中心一致        |
| **对称性**   | 质量分布尽量对称，减小惯性积              |

### 1.2 加工约束

| 项目       | 选择                         |
| ---------- | ---------------------------- |
| **主加工方式** | 3D 打印（FDM / 光固化）  |
| **补充加工**   | CNC（金属件、高精度件）  |
| **材料**      | 待定，见 §2 材料选型     |

---

## 二、材料选型

### 2.1 3D 打印材料（初版建议）

| 用途         | 推荐材料       | 特性                     |
| ------------ | -------------- | ------------------------ |
| 机架主体     | PLA / PETG     | 易打印、成本低、强度够   |
| 云台结构件   | PETG / ABS     | 韧性好、耐冲击           |
| 桨罩 / 护壳  | PLA / TPU      | 轻量、可柔性（TPU）      |
| 高负荷关节   | 尼龙 / 碳纤增强 | 强度高、耐磨（可选）     |

### 2.2 金属件（如采用 CNC）

| 用途         | 推荐材料       |
| ------------ | -------------- |
| 云台转轴     | 铝合金 6061    |
| 电机安装板   | 铝合金 2mm+    |
| 关键紧固件   | 不锈钢 M2/M3   |

---

## 三、关键尺寸约束

### 3.1 重心与推力点

```
        ↑ Z (竖直向上)
        │
        │    ● 重心 (CG)
        │    │
        │    │  h_cg   （重心到推力点垂直距离，需 > 0）
        │    │
        │    ▼ 推力作用点 (CP) ← 云台旋转中心
        │
        └──────────────────→
```

- **h_cg**：重心到推力点垂直距离，建议初版 50–150 mm
- **横向偏心**：尽量控制在 5 mm 以内

### 3.2 共轴反桨单元

| 参数           | 初版建议     | 说明                     |
| -------------- | ------------ | ------------------------ |
| 桨直径         | 5–7 inch     | 视推力与效率需求         |
| 上下桨间距     | 待仿真/试验  | 过小易干扰，过大增高度   |
| 桨罩直径       | 略大于桨径   | 兼顾保护与气流           |

### 3.3 二轴云台

| 参数           | 初版建议     | 说明                     |
| -------------- | ------------ | ------------------------ |
| 俯仰角范围     | ±15°–±25°    | 满足姿态控制需求         |
| 滚转角范围     | ±15°–±25°    | 同上                     |
| 偏转速度       | 待电机选型   | 需满足控制带宽           |
| 减速比         | 待定         | 兼顾力矩与响应           |

## 四、子系统设计要点

### 4.1 机架

- **形式**：管架 / 板架 / 混合，便于 3D 打印
- **安装点**：电池舱、飞控舱、云台基座、共轴单元
- **强度**：校核悬停 + 最大偏转力矩下的应力

### 4.2 共轴反桨单元

- **上下电机**：同轴安装，轴系精度要求高
- **桨罩**：防护与整流，避免吸入异物
- **散热**：电机散热路径设计

### 4.3 二轴云台

- **轴系**：俯仰轴 + 滚转轴，正交布置
- **电机**：无刷减速电机，需选型计算
- **限位**：机械限位 + 软件限位

### 4.4 电池与载荷布局

- **电池**：尽量上置，配合重心要求
- **飞控/IMU**：靠近重心，减少振动
- **线缆**：走线不干扰云台转动

## 五、设计流程与里程碑

| 阶段       | 工作内容                         | 交付物             |
| ---------- | -------------------------------- | ------------------ |
| **M0：概念** | 草图、尺寸估算、材料初选         | 概念草图、参数表   |
| **M1：详细设计** | 机架、共轴单元、云台 CAD 建模   | 零件与装配模型     |
| **M2：工程图** | 出图、BOM、加工说明              | 图纸、BOM          |
| **M3：样机** | 打印/加工、装配、重心测量        | 可装配样机         |

## 六、待定事项

3. **整机重量预算**：用于推力/功率校核
4. **标准件**：螺丝、轴承、传动件规格统一
