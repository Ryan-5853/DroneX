# 机架设计

## 功能

机架承载电池、飞控、载荷，并与共轴推进单元、二轴云台连接，是整机结构的骨架。外形类似**火箭**，细长，底部由起落架支撑。

## 已确认

- **形式**：管架为主
- **外形**：火箭式、细长
- **起落架**：底部支撑，具体形式待定

## 设计要点

- 管材：碳纤管 / 铝管 / 3D 打印分段管，便于加工与装配
- 保证重心在推力点之上（h_cg > 0）
- 电池舱、飞控舱沿轴向布置，配合细长外形
- 云台基座与共轴单元安装点刚性足够
- 起落架：着陆稳定性、触地面积、收纳/展开方式

## 待细化

- 管径、壁厚、接头形式（待推进单元/云台尺寸确定）
- 具体零件清单与装配顺序
- 强度校核
- 起落架支腿形式、展开角度
