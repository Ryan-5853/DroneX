# 共轴反桨推进单元

## 功能

上下两层反向旋转的螺旋桨，向下产生推力，自然抵消反扭矩，作为整机唯一推力源。

## 已确认

- **选型**：现成共轴产品 [淘宝 871597014800](https://item.taobao.com/item.htm?id=871597014800)
- **桨罩**：暂不定

## 设计要点

- 到货后测量：桨径、上下桨间距、总高度、安装孔位、重量、标称推力
