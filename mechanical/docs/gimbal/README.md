# 二轴云台

## 功能

俯仰轴 + 滚转轴，将推力矢量偏转，实现姿态控制与水平加速度。由两个**减速电机**直接驱动。

## 已确认

- **驱动方式**：减速电机直驱（无舵机、无步进）
- **选型**：[淘宝 733728260044](https://item.taobao.com/item.htm?id=733728260044)

## 设计要点

- 轴系正交布置，俯仰 + 滚转
- 到货后测量：扭矩、减速比、转速、安装尺寸、重量
- 机械限位与软件限位
- 线缆走线不干扰转动（中空轴或柔性走线）

## 到货后待补充

- 电机型号、额定电压/电流
- 减速比、堵转扭矩、空载转速
- 安装孔位、轴径
- 角范围建议（或按机械限位实测）
