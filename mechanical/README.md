# DroneX 机械设计子系统

## 概述

本目录包含 DroneX 可回收火箭式无人机的机械设计内容，包括机架、共轴推进单元、二轴云台等结构与装配设计。加工方式以 **3D 打印** 为主，必要时采用 CNC 加工。

## 设计目标

- **机架**：管架为主，火箭式细长外形，底部起落架，兼顾重心/推力点位置
- **共轴反桨单元**：现成共轴产品，到货后补充规格与安装接口
- **二轴云台**：俯仰轴 + 滚转轴，高端无刷舵机驱动
- **电池/载荷布局**：保证重心在推力点之上，满足倒立摆动力学要求（整机 ≤ 1 kg）

## 目录结构

```
mechanical/
├── docs/              # 机械设计文档
│   ├── DESIGN_PLAN.md # 设计计划与关键参数
│   ├── frame/         # 机架相关文档
│   ├── coax_unit/     # 共轴推进单元文档
│   └── gimbal/        # 二轴云台文档
├── cad/               # CAD 源文件（Fusion 360 / FreeCAD / STEP）
├── models/            # 导出模型（STL、STEP，用于 3D 打印与仿真）
├── drawings/          # 工程图（装配图、零件图）
└── bom/               # 物料清单（BOM）
```

## 输出物

| 类型       | 说明                         |
| ---------- | ---------------------------- |
| CAD 模型   | 零件与装配体源文件           |
| 工程图     | 装配说明、零件标注、加工图   |
| BOM        | 物料清单，含规格与数量       |
| STL/STEP   | 3D 打印用模型、仿真/外协用   |
