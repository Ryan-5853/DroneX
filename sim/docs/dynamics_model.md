# DroneX 动力学模型说明

仿真用动力学模型的坐标系、方程与参数约定。与项目主计划 [PLAN.md](../docs/PLAN.md) 第二章对应并细化。

---

## 1. 坐标系定义

### 1.1 惯性系 {I}

- 原点：地面固定点
- 轴：`Z_I` 向上，`X_I`、`Y_I` 水平（通常取北东地或前右地等约定）
- 用途：描述机体位置、速度、重力方向

### 1.2 机体系 {B}

- 原点：机体重心 (CG)
- 轴：随机体旋转，`Z_B` 通常指机体下方（推力默认方向）
- 用途：描述角速度、力矩、推力在机体中的分量

### 1.3 推力系 {T}

- 随云台偏转，描述推力矢量方向
- 云台无偏转时与机体系 Z 轴对齐
- 云台偏转 (α, β) 将推力从 `Z_B` 偏转至空间方向

### 1.4 旋转约定

- 姿态表示：四元数或欧拉角（ZYX 或 312 等，待定）
- 云台角：俯仰 α、滚转 β，定义与正方向待与机械设计对齐后补充

## 2. 运动方程

### 2.1 平动（牛顿第二定律）

```
m · a_I = F_thrust + m · g + F_disturbance
```

- `m`：机体质量 (kg)
- `a_I`：惯性系下的加速度 (m/s²)
- `F_thrust`：推力矢量，由云台角 (α, β) 与推力幅值 T 决定
- `g`：重力加速度向量，如 [0, 0, -g]
- `F_disturbance`：扰动力（风等）

`F_thrust` 需从推力系转换到惯性系，通过机体姿态与云台角实现。

### 2.2 转动（欧拉方程）

```
I · ω̇ + ω × (I · ω) = τ_thrust + τ_disturbance
```

- `I`：绕重心的转动惯量矩阵 (kg·m²)
- `ω`：机体系下的角速度 (rad/s)
- `τ_thrust`：推力产生的力矩，由推力作用点相对重心的力臂产生
- `τ_disturbance`：扰动力矩

**倒立摆特性**：重心在推力点之上时，推力偏转产生恢复力矩，但开环不稳定，必须闭环控制。

### 2.3 推力矢量与云台

云台偏转 (α, β) 使推力产生：

1. **姿态控制力矩**：推力线不通过重心，形成力臂
2. **水平加速度**：推力具有水平分量

推力幅值 T 与云台角 (α, β) 由控制分配模块根据期望力/力矩计算。

## 3. 参数表（占位）

以下参数待动力学推导与机械设计确定后填充。

### 3.1 刚体参数

| 符号 | 含义           | 单位  | 初值/备注     |
| ---- | -------------- | ----- | ------------- |
| m    | 质量           | kg    | 待定          |
| Ixx  | X 轴转动惯量   | kg·m² | 待定          |
| Iyy  | Y 轴转动惯量   | kg·m² | 待定          |
| Izz  | Z 轴转动惯量   | kg·m² | 待定          |
| Ixy, Ixz, Iyz | 惯量积 | kg·m² | 通常假设为 0  |

### 3.2 几何参数

| 符号   | 含义               | 单位 | 初值/备注          |
| ------ | ------------------ | ---- | ------------------ |
| r_cg   | 重心位置（机体系） | m    | 待定               |
| r_thrust | 推力作用点相对重心 | m    | 重心在推力点之上，Z 分量为负 |

### 3.3 推进器参数

| 符号 | 含义             | 单位  | 初值/备注           |
| ---- | ---------------- | ----- | ------------------- |
| k_T  | 推力系数         | N/(rad/s)² 或类似 | 待测或经验公式     |
| η_coax | 共轴干扰系数   | -     | 上/下桨效率比，待测 |

### 3.4 云台参数

| 符号   | 含义           | 单位   | 初值/备注      |
| ------ | -------------- | ------ | -------------- |
| α_lim  | 俯仰角限位     | rad    | 待定           |
| β_lim  | 滚转角限位     | rad    | 待定           |
| τ_gimbal | 云台响应时间常数 | s    | 电机/减速器决定 |

---

## 4. 与仿真的对应

- 平动方程 → Simulink 积分块或 MATLAB ODE
- 转动方程 → 同上，注意四元数微分方程或欧拉角奇点
- 推力矢量 → 由 `(T, α, β)` 计算 `F_thrust`、`τ_thrust` 的子模块
- 参数 → 统一在 `matlab/scripts/` 或 Workspace 中配置
