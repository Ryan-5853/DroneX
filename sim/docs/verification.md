# DroneX 仿真验证标准

仿真子系统的验证指标与典型用例，用于评估动力学模型和控制算法的正确性与性能。详见 [SIMULATION_PLAN.md](SIMULATION_PLAN.md)。

---

## 1. 验证目标

- 确认动力学模型与理论一致
- 确认控制器能稳定倒立摆式无人机
- 为实机参数调试提供初值依据

---

## 2. 姿态控制验证

### 2.1 阶跃响应

| 指标       | 含义                     | 目标（待定）       |
| ---------- | ------------------------ | ------------------ |
| 上升时间   | 达到 90% 设定值的时间    | 例：< 1.0 s        |
| 超调量     | 最大 overshoot           | 例：< 20%          |
| 调节时间   | 进入 ±5% 误差带的时间    | 例：< 2.0 s        |
| 稳态误差   | 阶跃后的稳态偏差         | 例：< 2°           |

### 2.2 测试用例

- 俯仰角阶跃：0° → 10°
- 滚转角阶跃：0° → 10°
- 偏航角阶跃（若实现）：0° → 15°

---

## 3. 悬停验证

### 3.1 指标

| 指标         | 含义                     | 目标（待定）       |
| ------------ | ------------------------ | ------------------ |
| 位置误差     | 稳态水平/垂直偏差        | 例：< ±0.5 m       |
| 速度误差     | 稳态速度幅值             | 例：< ±0.2 m/s     |
| 姿态偏差     | 俯仰/滚转稳态角          | 例：< ±3°          |

### 3.2 测试用例

- 定点悬停：初始位置 (0, 0, 5 m)，目标 (0, 0, 5 m)，仿真 30 s
- 水平平移后悬停：从 (0, 0, 5 m) 到 (2, 0, 5 m)，观察到位与稳态

---

## 4. 抗扰验证

### 4.1 指标

| 指标         | 含义                         | 目标（待定）       |
| ------------ | ---------------------------- | ------------------ |
| 恢复时间     | 扰动后回到误差带内的时间     | 例：< 3.0 s        |
| 残余误差     | 扰动撤除后的稳态偏差         | 例：< 5% 设定值    |

### 4.2 测试用例

- 阶跃风扰动：水平恒风 2 m/s 持续 5 s 后撤除
- 脉冲力矩扰动：短时施加俯仰/滚转力矩，观察恢复

---

## 5. 记录与报告

每次验证应记录：

1. 仿真参数（质量、惯量、控制增益等）
2. 典型曲线（姿态、位置、控制量）
3. 与上述指标的对比结果
4. 结论：通过/不通过，及改进建议

输出保存至 `sim/data/results/`，并汇总到仿真报告中。
